# 布隆过滤器

### 定义：

用于检索一个元素是否在集合中

存储时为一个元素分配多个二进制位 （所以存在误判）

查到一个元素存在 => 可能存在

查到一个元素不存在 => 一定不存在



### 用途：

放在数据库外，作为快速查询的缓存，如果在布隆过滤器中不存在，则不需要进入db查询



### 案例	

- 比特币网络
- 分布式系统
- Redis缓存
- 垃圾邮件、评论过滤



# LRU Cache

### 缓存机制：

1)新数据插入到链表头部；
2)每当缓存命中（即缓存数据被访问），则将数据移到链表头部；
3)当链表满的时候，将链表尾部的数据丢弃。
